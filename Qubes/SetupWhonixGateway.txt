Fresh Whonix Gateway
  Update/upgrade everything
  Download current i2pd release from GitHub using "get-latest-i2pd-sh"
  Copy /tmp/i2pd.deb to gateway template
  Run "setup-whonix-gw-i2pd.sh" in gateway template.
  Let the GW sit for a few minutes while router reseeds, addressbooks are downloaded, and tunnels are built.
